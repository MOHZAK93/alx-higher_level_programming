The ``MyList`` module

Using ``print_sorted``
---------------------

>>> MyList = __import__("1-my_list").MyList
>>> my_list = MyList()

>>> my_list.append(1)
>>> my_list.append(5)
>>> my_list.append(3)
>>> print(my_list)
[1, 5, 3]

>>> print(my_list)
